smriprep.cli.run.build_workflow(opts, retval)[source]¶

Create the Nipype Workflow that supports the whole execution graph, given the inputs. All the checks and the construction of the workflow are done inside this function that has pickleable inputs and output dictionary (retval) to allow isolation using a multiprocessing.Process that allows smriprep to enforce a hard-limited memory-scope.
